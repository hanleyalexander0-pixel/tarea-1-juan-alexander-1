# Actividad 3 – Flujo de Control Parte 2

## Descripción
Este programa permite ingresar las 4 calificaciones de una cantidad n de estudiantes, calcular el promedio y determinar si el estudiante aprobó o reprobó.

El programa utiliza estructuras repetitivas (for) y condicionales (if-else) en C++.

## Funcionamiento
1. Se ingresa la cantidad de estudiantes.
2. Se introducen las 4 notas de cada estudiante.
3. Se calcula el promedio.
4. Se determina el estatus:
   - Aprobado (promedio >= 70)
   - Reprobado (promedio < 70)
5. Se muestran los resultados en formato de tabla.

## Ejemplo de salida

Estudiante          Nota1  Nota2  Nota3  Nota4  Promedio  Estatus
-----------------------------------------------------------------------
Juan Perez          80     75     90     85     82.5      Aprobado
Maria Lopez         60     65     58     62     61.25     Reprobado
